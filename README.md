# Aniimer – Anime Blog App

A mobile-first anime blog interface built with pure HTML and CSS.

This project focuses on layout structure, responsive design, and clean UI organization without JavaScript.

## Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Mobile-First Approach

## Features

- Mobile-first layout
- Responsive design (mobile and desktop support)
- Fixed header with adaptive navigation
- Interactive dropdown menus using <details>
- Responsive grid layout for anime cards
- Organized folder structure
- Clean and scalable CSS architecture

## Purpose

This project was built as part of my front-end development journey to strengthen:

- Layout structuring
- UI consistency
- Mobile-first design
- Project organization for production environments
